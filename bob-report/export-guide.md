# IBM Bob Report Export Guide

This folder contains exported IBM Bob session reports generated by RepoPilot.

## What Is a Bob Report?

A Bob Report is a JSON file capturing every IBM Bob AI session that occurred
during a RepoPilot run. Each session includes:

- **Session ID** — unique identifier for the Bob call
- **Type** — one of: `repo-summary`, `doc-generation`, `test-scaffold`, `pr-review`
- **Prompt** — the exact prompt sent to IBM Bob
- **Response** — the exact output IBM Bob returned
- **Timestamp** — when the session occurred

---

## How to Export a Report

### Option 1 — From the Dashboard (Recommended)
1. Open RepoPilot in your browser.
2. Navigate to the **⚡ Bob Sessions** tab.
3. Click **📦 Export Bob Report**.
4. The report is saved to `/bob-report/sessions/` automatically.

### Option 2 — Via API
```bash
curl -X POST http://localhost:5000/api/report/export \
  -H "Authorization: Bearer your_api_secret"
```

### Option 3 — List All Saved Reports
```bash
curl http://localhost:5000/api/report/list \
  -H "Authorization: Bearer your_api_secret"
```

### Option 4 — View Live Sessions (not yet exported)
```bash
curl http://localhost:5000/api/report/live \
  -H "Authorization: Bearer your_api_secret"
```

## Report File Format

Reports are saved as timestamped JSON files:
bob-report/
└── sessions/
├── bob-session-2024-06-15T14-32-00-000Z.json
├── bob-session-2024-06-15T18-45-10-000Z.json
└── session-example.json   ← example file for reference
Each file contains:

```json
{
  "reportId":     "uuid",
  "projectName":  "RepoPilot",
  "exportedAt":   "ISO timestamp",
  "totalSessions": 4,
  "sessionTypes": {
    "repo-summary":   1,
    "doc-generation": 1,
    "test-scaffold":  1,
    "pr-review":      1
  },
  "sessions": [ ...session objects... ]
}
```
